Player 1 :

Up		->	Z

Down		->	S

Left		->	Q

Right		->	D

Start 1		->	K

2		-> 	L

Player 2 :

Up		->	Up Arrow

Down		->	Down Arrow

Left		->	Left Arrow

Right		->	Right Arrow

Start 1		->	Numpad 2

2		-> 	Numpad 3

Emulator :

Reset		->	SpaceBar
